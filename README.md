# TaskFlow API

A role-based task management backend application built with Node.js, Express.js, TypeScript, and MongoDB.

## Project Goal

This project is designed to learn and implement production-style backend concepts beyond authentication.

## Learning Objectives

- Role-Based Access Control (RBAC)
- Authorization
- Pagination
- Search & Filtering
- File Uploads
- Cloud Storage
- Advanced Validation
- Layered Architecture

## Tech Stack

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT
- bcrypt

## Architecture

Route
↓
Controller
↓
Service
↓
Repository
↓
Database
